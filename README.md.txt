# 🎯 Hit or Miss

A full-stack project with:
- **Backend:** Python (Flask) — Implements algorithms and handles API requests.
- **Frontend:** React + Tailwind CSS — Interactive UI for visualizing and using the backend features.

## ⚙️ Installation

### **Backend Setup**
```bash
cd backend
python -m venv venv
venv\Scripts\activate  # Windows
pip install -r requirements.txt
python app.py


***Frontend Setup***
cd frontend
npm install
npm start

Then open http://localhost:3000 in your browser.

🚀 Usage
Start the backend server.

Start the frontend server.

Use the frontend UI to interact with the backend algorithms in real time.

🛠 Technologies Used
Backend: Python, Flask

Frontend: React, Tailwind CSS

Testing: Jest, React Testing Library
